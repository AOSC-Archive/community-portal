2012-02-03更新：AMD64版本
增加实验性特征：Dropbox自动备份脚本，fcron（功能更强大，用于代替cron，因而去除cron）
去除：phpMyAdmin（安装脚本稍后放出，没有包括在系统中）

Dropbox自动备份脚本为/usr/local/bin/dropboxd，请参照http://shuyz.com/auto-backup-openshift-to-dropbox-1.html对该文件进行修改

增加Typecho和Drupal的脚本，为
autoinstaller typecho和autoinstaller drupal

Zend Server没有进行初始化，请参照一楼访问的Zend Server后台进行初始化，否则无法享受Zend Server（PHP正常）

其他用户名密码不变

CloneZilla镜像改名zend-linux-remix-amd64

下一次更新将改名Anthon-CentralPoint（有中文会有一些问题的），并且有会停止i386分支

地址：http://pan.baidu.com/share/link?shareid=253660&uk=1596170030记得大概在半年以前，安同团队曾经有开发服务器版本的想法，不过貌似自从从SUSE Studio发行版换到Debian以后服务器分支貌似就没有消息了。说实话我当时还在想等安同服务器版本成熟然后我们学生会就有中学生自己开发的服务器可以用了，现在感觉好失望。

今天下午研究了一下安同，发现我是没可能为开发团队做出核心贡献了，所以花了一些时间做了一个符合我的口味的服务器系统，暂时命名为"Zend Linux Remix"，已废弃，改名为"CentralPoint"

基于Debian Squeeze（没有用testing/unstable是因为与我的核心之一Zend Server Community Edition有依赖冲突）
默认安装Apache2, MySQL, Zend Server Communtiy Edition以及打过Zend补丁的PHP（我自己是学PHP和研究相关运维的，这是我优化性能的法宝）
安装了Sun JRE, Zend Java Bridge, Zend Guard Loader
支持通过我手工编写的bash脚本一键安装常用Web应用
（我自己是学PHP得，对这一块比较清楚）

内置的账户和密码：
root：已禁用
默认用户：tux，密码：linux
MySQL默认root密码：toor
Zend Server后台密码：admin

登陆技巧：
SSH监听TCP端口22
Zend Server采用Web面板，端口为10081（HTTP）或者10082（HTTPS）

安装Web应用：
执行"autoinstaller [应用名]"以将应用自动下载并安装到/var/www（默认站点的目录），只能安装一个，否则会有冲突
支持的应用
dedecms-full：织梦CMS完整版
discuzx：Discuz! X
dokuwiki：DokuWiki
emlog：EMlog
hdwiki：HDWiki
mediawiki：MediaWiki
phpbb：phpBB
phpwind：PHPWind
wordpress：WordPress

其他技巧：
输入形如"IP/tz.php"来查看服务器信息
使用HTTPS登陆Zend Server后台会提示证书无效，无视即可，不过这意味着你无法在访问远端运行的操作系统时受到防中间人攻击的保护，这种情况建议使用受SSH端口映射保护HTTP连接访问后台
在虚拟机中请自行设置端口映射
虚拟机中运行时，通过SSH连接会比直接在窗口中敲要爽，因为可以复制粘贴（强烈推荐Bitvise SSH Client，还自带基于SSH的远程/本地双向文件管理器，连客户机文件共享插件都省了）

CloneZilla安装（会完全擦除硬盘数据）：首先下载包含磁盘镜像的CloneZilla ISO，用它引导计算机，进入CloneZilla后在"挂载再生龙镜像文件的目录"那一屏（前提是你之前选择语言那一屏选的是简体中文，其他语言意会即可）选择"enter_shell 进入命令列，自行手动调整"选项，在进入shell后输入"ln -s /live/image/zend-linux-remix/ /home/partimag"（可以tab自动补齐），成功后输入"exit"然后再按照标准再生龙的克隆方式进行克隆（注意 本镜像一共有3个分区，如果打算手动调整分区，则要注意：第一个分区为/boot，不得小于200M，第二个分区为/，不得小于13G，第三个分区为swap。不得小于770M）

下载地址（有OVF Applet和CloneZilla ISO，不过前者较大）：http://pan.baidu.com/share/link?shareid=253659&uk=1596170030
