实战：CentralPoint部署Typecho博客程序

本帖最后由 butangmucat 于 2013-4-6 01:12 编辑

原文：http://wiki.anthonos.org/?p=172

本指南假定你已经安装好了新发布的0.99pre系统和cp-update_1.0.0~vanessa-1，否则请参照安同安装指南安装0.99pre并到这里下载安装cp-update_1.0.0~vanessa-1
本例中服务器为端口映射后的虚拟机，地址为127.0.0.1:22，用户为tux
http://wiki.anthonos.org/wp-content/uploads/2013/04/%E6%8D%95%E8%8E%B7-298x300.png

在提示输入密码“linux”之后，系统会自动打开一个shell窗口和一个sftp commander。
http://wiki.anthonos.org/wp-content/uploads/2013/04/%E6%8D%95%E8%8E%B71-300x179.png

执行命令
CentralPoint使用的是MariaDB数据库来提供MySQL相关功能，目前除CP以外OpenSUSE Slackware Arch Magiea也已经使用MariaDB代替了MySQL
连接到MariaDB执行命令
# mysql -u root -p

接着会提示输入密码，输入”toor”
添加新用户和对应的数据库INSERT INTO mysql.user(Host,User,Password) values(“localhost”,”typecho”,password(“password”));


CREATE DATABASE typecho;

FLUSH PRIVILEGES;

这样就建立了typecho用户，密码password，并授予该用户typecho数据库的全部权限
输入“exit”来离开MySQL客户端

安装Typecho获取并安装Typecho源程序# axel http://typecho.googlecode.com/files/0.8\(10.8.15\)-release.tar.gz

cp -r build/* /var/www

配置目录权限执行命令
# chmod -R 777 /var/www

初始化Typecho注意：本例中虚拟机的80端口被映射为主机的10080端口，实际操作时请根据情况自行修改相关URL信息等主机浏览器访问http://127.0.0.1:10080，如果出现下面的界面，你离成功就不远了！http://wiki.anthonos.org/wp-content/uploads/2013/04/%E6%8D%95%E8%8E%B76-300x240.png
