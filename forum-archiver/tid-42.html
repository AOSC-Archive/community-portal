Grub2 引导硬盘上的ISO文件

安装Linux系统可以先将iso文件通过软件放到U盘中，然后从U盘启动进行安装，如果手头上没有U盘，那么我们可以直接引导硬盘中的ISO文件。
通过引导菜单直接引导ISO文件已经不是新鲜事了，在Windows下可以使用EasyBCD软件添加ISO引导项，当然在Linux下也可以直接引导iso文件。
首先准备一个iso文件，目录最好不要放的太深。执行如下命令查看下硬盘的位置：
sudo fdisk -l
比如iso文件存储在/dev/sda7上。
打开grub文件(vi不熟悉可以使用gedit等文本编辑软件)：
sudo vi /boot/grub/grub.cfg
在后面加上如下内容：
menuentry “Ubuntu 12.10 LiveCD” {
set root=(hd0,7)
loopback loop /…/ubuntu-12.10-desktop-i386.iso
linux (loop)/casper/vmlinuz boot=casper iso-scan/filename=/…/ubuntu-12.10-desktop-i386.iso   noprompt noeject
initrd (loop)/casper/initrd.lz
}
红色的7对应的就是/dev/sda后面的数字（比如7）,另外两个红色的就是iso的文件路径。
