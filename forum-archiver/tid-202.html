[On Air]安同CentralPoint新版构建

本帖最后由 butangmucat 于 2013-3-24 23:30 编辑

本版代号：Vanessa

2013-3-24更新：支持图形界面！！！







基本系统，基于Wheezy使用systemd，带有Zend Server 6.0 Free Edition, PHP 5.4, Ruby 1.8, Perl 5.14, Python 2.7.3 Mono 2.0

Apache支持PHP, Ruby (Rails), Perl, Python, ASP.NET 2.0

添加图形界面，通过startx手动开启，正在研究远程桌面解决方案
图形界面包含openbox+tint2桌面（详见：http://bbs.chinaunix.net/thread-1969378-1-1.html），Midori浏览器，Roxterm终端，Geany文件编辑器，考虑添加Conky，从Parted Magic黑配置文件，考虑添加fcitx
图形界面使用Google Droid字体提供中文支持，使用qtcurve主题+oxygen图标

目测开机时间在25秒以内
