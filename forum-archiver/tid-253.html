安同最终公测版意见以及Bug反馈帖

本帖最后由 jeffbaichina 于 2013-4-1 11:18 编辑

从这里开始，不定期整理更新。请规范发帖，方便开发者。

我们可能需要你的帮助，谢谢：
请帮助我们测试两个特性：打开文件管理器（无问题）、收起应用菜单（无问题）、打开默认桌面设置（桌面上右键，无问题）
已证实这些特性没有问题，感谢@宝先森

整理：
× BUG：系统设置-多媒体，卡死，用kdesudo systemsettings命令启动时不会
   已解决：下一个更新包将解决问题。PS: 同样修复没有声音的问题
   技术细节：anthon-updated: + sudo gpasswd -a "$(whoami)" lpadmin
                                                + sudo gpasswd -a "$(whoami)" audio
                                                + sudo gpasswd -a "$(whoami)" users
                                                + sudo gpasswd -a "$(whoami)" bluetooth
                                                + sudo alsactl init
× 建议：开机后设置数字键盘自动开启（每一次输密码都要摁一下很烦人）
   已解决：下一个更新包将自动更改配置
   技术细节：anthon-updated: + cat > /home/"$(whoami)"/.kde/Autostart/numlockx << EOF
                                                + #!/bin/sh
                                                + numlockx on
                                                + EOF
                                                + sudo chmod 755 /home/"$(whoami)"/.kde/Autostart/numlockx
× BUG: 总是会有烦人的PC Speaker蜂鸣
   已解决：下一个更新包中将屏蔽该设备
   技术细节：anthon-updated: + sudo echo "blacklist pcspkr" > /etc/modprobe.d/nobeep.conf
× BUG: 蓝牙似乎没有正常工作
   已解决：下一个更新包中将解决该问题
   技术细节：anthon-updated: + sudo apt-get install bluetooth
× BUG:用户新建之后似乎将使用/bin/sh而非/bin/bash，而且用户没有存在于应该存在的组
   已解决：下一个更新包中将解决（但用户需要新建用户）
   技术细节：/etc/default/useradd: + GROUP=100, 111, 109, 29
                                                      + SHELL=/bin/bash
× 建议：设置一个随时可以看时间的东西
   用户操作：请自行更改，在桌面上右键，选择新建空面板即可自行定制。
× BUG: 登录后黑屏。
   已解决，操作失误
× BUG: 登录画面似乎不如上次流畅
   正常现象：我们添加了新的服务
× BUG: 文件管理器需要30秒才能打开
   无法解决：未能重现
× BUG: 无法收起应用菜单
   无法解决：未能重现
× 建议：正式版去掉prelink吧，对速度影响不大，技术陈旧可能与multiarch冲突，而且难以维护，不适合新手。
   已应用于下个更新包
   技术细节：anthon-update: - sudo prelink -amvR
                                             + sudo prelink -ua
                    anthon-updated: - sudo prelink -amvR
                                               + sudo prelink -ua
