CentralPoint 0.99 Pre-release "Vanessa"发布贴

本帖最后由 butangmucat 于 2013-4-4 22:45 编辑

在跳票一周之后，代号"Vanessa"的CentralPoint正式进入Pre-release阶段，版本号为0.99
｛这是一个向女神致敬的版本｝

特性：
PHP 5.4
Zend Server 6.0.1
Ruby 1.9.1
Python 2.7
Phusion Passenger 3.0.19
ASP.NET 2.0
MariaDB 5.5
NoMachine NX 3.5
从Gentoo那里偷来的彩色终端bashrc
从http://forum.anthonos.org/forum.php?mod=viewthread&tid=27偷来的vimrc

注意：
Zend Server后台：http://127.0.0.1:10081 首次使用配置时请选择“Development“模式而不是“Production“模式，否则会从Free Edition变成Enterprise Trail且无法解决，只有重装安同
Phusion Passenger的Python支持比较复杂，请参见：http://wiki.dreamhost.com/Passenger_WSGI
系统默认为弱密码，如用于生产环境请务必修改！
MySQL支持由与之完全兼容的MariaDB提供，但是系统服务依然为mysql，且MariaDB软件包与MySQL有冲突，不能共存，卸载并更换MySQL请用aptitude，具体参见：https://mariadb.org/
NoMachine NX需要手动配置，请参见：http://www.ovear.info/post/19

用户名密码
系统默认用户：tux 密码：linux
MariaDB默认root密码：toor
如果使用UltraISO或者Unetbootin制作U盘安装CP，在指定“光盘”时请输入U盘的设备名，比如sdb1
