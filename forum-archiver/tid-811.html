OS2 最终架构预览发布！

本帖最后由 jeffbaichina 于 2014-1-25 07:43 编辑

OS2 在经过 6 个月的开发后终于走到了旅程的终点，系统结构和技术已经基本定型。但 OS2 只是我们社区第二代发行版的技术基础，接下来我们要做什么呢？

- 分线，AnthonOS / IcenowyLinux / CentralPoint 将以此基础开始构建，制作出三种不同用途的发行版：
  AnthonOS - 一个激进的滚动更新桌面发行版
  IcenowyLinux - 一个稳定的桌面发行版
  CentralPoint - 一个轻量化的，易于部署的小型 / 家庭服务器发行版
  我们开发组将和社区成员一齐努力，在 OS2 的技术基础上实现三种用途的发行版，并提供尽可能好的体验和实用性。
- 准备 OS3，OS3 是安同开源社区的第三代发行计划，将在 OS2 的基础上进行改进。AnthonOS 在暑假后将成为 OS3 的基础。

OS2 即将在开发组内部测试完成后发布，在下载放出之前，我先在这里介绍下 OS2 的基本技术特征（抱歉，暂时用户体验还不是我们要研究的话题）。

- OS2 是一个以 Linux From Scratch 项目和相关书籍作为参考实现的又一个独立发行版，而不再是 OS1 时期的 Debian 衍生版本。
- OS2 采用 Debian 包管理，但是分包以 “套件” 作为单位，而不再细分任何软件包，减少安装软件的迷惑性。
- OS2 采用完全的 systemd 启动 / 初始化模式，以 Linux 的方式实现 Linux 的启动。
- OS2 提供了多种桌面环境的完整支持，KDE / XFCE / MATE / LXDE / Cinnamon / GNOME / Trinity / Awesome / GNUStep 等，甚至还支持 Deepin 桌面环境。
- OS2 实现了类似 Arch 社区的 AUR 非官方仓库，给有兴趣的用户更多提供个性化软件包的可能性。
- OS2 采用 32subsystem (32 位子系统) 实现 32 位软件包的兼容，安装 Kingsoft Office / Skype 只需轻按鼠标就能解决。
- OS2 的 Autobuild 包构建工具集实现了和包管理的隔离，写好配置，然后输入命令 autobuild 然后系统将自动完成依赖补齐、构建和打包，现已支持 DPKG 和 RPM 包管理。
- OS2, Steam, yeah! What?
- OS2 使用一个基础实现了三个发行版的发行，让 AnthonOS / IcenowyLinux / CentralPoint 之间实现了完整的二进制兼容，同时还减少了我们这个小型开发组同时构建并维护三个发行版的压力。

不过对于用户来说，这些事情暂时还没能体现出其优势…… (这和我有关系么？) 那么我在这里解释一下这些 “特性” 能给在不久后发布的三个发行版提供的便利，大概如下：
- 你现在可以在 15 秒内完成系统启动！
- 你现在可以玩到最新的正版游戏，还随时有限免！
- 有些软件我找不到诶，也许有人提供了呢 (开发组真没出息啊 23333333)！
- 现在妈妈终于不用担心我把一个下午都耗在安装 Skype 了！
- 我的笔记本退役了，当服务器用吧…… 哇，我都不用重装系统啦！
所以各位小白们请好好期待吧！我们一定努力不让你们失望。

经过多次脑洞大开的测试之后我们终于可以放出下载了。这里提供两个版本的下载：
- Live, 又称 OOBP (开箱即用预览版本)，提供了一个标准的 KDE 桌面环境并预装了一些桌面软件，软件包列表请看 传送门 ，建议愿意尝试的小白在虚拟机试用。
- Core，一个基本系统，只提供了基本的联网和启动设施，建议由经验的用户和开发者使用。这个版本一般用于系统快速部署而非日常使用。
