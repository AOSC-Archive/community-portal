今日论坛等站点下线的一点声明

今日论坛/Wiki/新闻站点下线，具体说，是被下线。这是一个来自所谓“天河联盟”的恶意 DDoS 攻击，导致服务器数据库崩溃（2001 - Disconnect）。我们社区对此组织从未进行过任何的攻击（包括人身以及站点等所有方面），而对方声明是本人与对方发生纠纷（下面具体说明）因此进行的报复行为。首先，谴责这种恶意的攻击行为，尤其是对于一个开源社区，这种行为显得尤为可耻。

下面对“纠纷”进行一下解释：
2013年11月9日，本人在 QQ 收到信息，来自 ccs2007，称邀请本人加入 Windows Beta Collection（一个 Windows Beta 版本收藏站点）。而当本人问及其资源来源时，对方表示来自外国的同类网站 BetaArchive。而 BetaArchive 论坛所有者 mrpijey 在此帖明确表示：
FTP Server Contents
- Mirroring the contents of the FTP server should be for personal use only and distribution to anyone outside of the forum is forbidden. The reasons for this can be discussed with you if requested.
那么很明显这个组织正在违反这个协议（许可）。本人首先拒绝了加盟请求（以个人名义而非此社区），其次对这种行为进行文明的批评（无攻击），然后退出了他们的讨论群组。接下来发生了如下的事情。大家可以到这个贴吧的帖子围观：
http://tieba.baidu.com/p/2699932636

稍加观察也许可以发现他们不是第一次进行攻击。

我代表社区要求对方立即道歉，如继续进行此类操作，对方将承担此行为的严重后果（不排除警方介入）。
