经过三个月的构建，我们的第二代系统 (a.k.a. :Next) 终于成型，现放出一个面向用户的技术预览，命名为 Out-of-Box Preview 1 (oobp)。该发行带有基本的生产、娱乐以及开发环境，但对于具体的用户体验如 UX 设计并没有进行具体的设计。

声明:
* 此发行作为 Pre-Beta 阶段放出，为不给初级用户带来过多麻烦，故提供 Live 预览而不提供安装功能；
* 此版本并非最终发行，可能存在大量的功能问题，请谅解；
* 我们不对此发行中 Contrib 或 Non-Free 部分的软件进行任何形式的支持；
* 开发将持续进行，今明两日开发组将进行 TODO List 讨论；

KDE 桌面



Mate 桌面

特性列表:
* Linux Live Kernel = 3.10.12 + aufs + AOSC Squeezed +reiser4
* KDE, Mate + XFCE
* Firefox, Chromium
* Audacious, VLC + MPlayer
* LibreOffice, GIMP
* PackageKit
* APT

关键特性目标:
* 无缝软件包更新以及安装，采用 PackageKit 实现
* 尝试多个桌面环境的配置共存以及迁移能力
* Live 内核测试
* 基本生产环境应用实验

已知问题:
* 由于特殊设计的音频架构，alsa-lib -> PulseAudio -> Jack (@zxd123314 如有错误请指正)，在 KDE 下很可能出现无声的情况。
* PackageKit 很可能依然存在权限问题。
(Still, more to come...)
